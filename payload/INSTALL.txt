NX Gallery prototype installation

1. Copy the archive's switch/nxgallery directory to the SD card's /switch directory.
2. Launch NX Gallery from the Homebrew Menu with full memory access.
3. If NX Torrent is not already configured, follow the full-screen QR setup.
4. Alternatively, rename telegram-bot.conf.example to telegram-bot.conf, replace the placeholder token, and optionally add chat=ID|Title lines for destinations that should always appear.
5. Start or add the bot in each desired Telegram chat, then use Y in the picker to discover pending chats.

When a newer stable GitHub release is available, NX Gallery shows a Minus Update
action in the bottom-left of the gallery. Press Minus or tap it, then restart the
app after the verified update finishes.

The app reads captures from /Nintendo/Album and never modifies them.
The real telegram-bot.conf is a credential. Do not publish or attach it to bug reports.
